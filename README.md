# Expense Tracker (Python)

A simple command-line expense tracker built using Python to help users record, store, and manage daily expenses.

## Features
- Add new expenses with amount and category
- View all recorded expenses
- Persistent storage using file handling
- Simple and user-friendly CLI interface

## Tech Stack
- Python
- File Handling
- Basic Data Structures

## Learning Outcomes
- Improved understanding of Python fundamentals
- Hands-on experience with real-world problem solving
- Practice with clean logic and modular code structure

## How to Run
1. Clone the repository
2. Run the Python file using: python expense_tracker.py
